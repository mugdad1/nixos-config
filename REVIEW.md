## 🖨️ HP Printing — wired (USB) + network (IP)

Web research (NixOS Wiki "Printing", Discourse, hplip docs) says: CUPS via
`services.printing`, HP drivers via `pkgs.hplip`, USB via `usb://` URI, network via
`socket://IP:9100` (or `hp:/net/NAME?ip=IP`), and Avahi/mDNS for auto-discovery.

**New file: `modules/core/printing.nix`** ✅ DONE (created + imported in `modules/core/default.nix`)
```nix
{pkgs, ...}: {
  # CUPS print server + HP drivers
  services.printing = {
    enable = true;
    drivers = with pkgs; [ hplip ];
    # openFirewall = true;  # only if you want to print TO/FROM other machines
  };

  # Auto-discover network/IP printers (mDNS). USB works without this.
  services.avahi = {
    enable = true;
    nssmdns4 = true;
    openFirewall = true;
  };

  # GUI printer manager
  programs.system-config-printer.enable = true;

  hardware.printers = {
    # --- Wired / USB HP printer ---
    # Plug it in, then: `lpinfo -v`  -> gives the usb://... URI
    #                  `lpinfo -m | grep -i hp` -> gives the model ppd
    ensurePrinters = [
      {
        name = "hp-usb";
        location = "Desk (USB)";
        deviceUri = "usb://HP/ENVY_$(...)_series?serial=XXXX";  # from lpinfo -v
        model = "drv:///hp/hpcups.drv/hp-envy_$(...)_series.ppd";  # from lpinfo -m
        ppdOptions = { PageSize = "A4"; };
      }

      # --- Network / IP HP printer ---
      # Use the printer's LAN IP (find it from router). socket:// is driver-less IPP.
      # Or for full HP features: deviceUri = "hp:/net/ENVY_...?ip=192.168.1.50";
      {
        name = "hp-network";
        location = "Network (IP)";
        deviceUri = "socket://192.168.1.50:9100";
        model = "drv:///hp/hpcups.drv/hp-envy_$(...)_series.ppd";
        ppdOptions = { PageSize = "A4"; };
      }
    ];
    ensureDefaultPrinter = "hp-network";
  };
}
```
**Steps after adding the module + `imports = [ ./printing.nix ]` in `modules/core`:**
1. `sudo nixos-rebuild switch`
2. `lpinfo -v` → copy the `usb://...` / `socket://...` line into `deviceUri`.
3. `lpinfo -m | grep -i hp` → copy the matching `drv:///hp/...ppd` into `model`.
4. Rebuild. Printers appear in `system-config-printer` and apps.
> Note: some HP models need the proprietary plugin — `hplipWithPlugin`
> (already allowed since `nixpkgs.config.allowUnfree = true`). For most modern
> HP "eSeries"/OfficeJet, `hplip` alone is enough.

## Web-research takeaways (best practices)

- Hyprland 0.55+ deprecated hyprlang for **Lua** — your manual `xdg.configFile`
  Lua approach is correct and gives full control (Home Manager's built-in Lua
  support is an alternative but less flexible for your gruvbox interpolation).
- NixOS-unstable is a fine daily-driver (you already use it); rollbacks make
  breakage cheap.
- Printing on NixOS = `services.printing` (CUPS) + `hardware.printers.ensurePrinters`.
- Hardening references: `nix-mineral`, notashelf's systemd-sandboxing posts,
  madaidan's Linux hardening guide. Defense-in-depth, not single silver bullets.
